# Hello PCF Tile
An example no-op tile contained in one metadata file to get more familiar with [PCF Tile structure](https://docs.pivotal.io/tiledev/2-1/tile-basics.html). This is in opposition to how [kiln](https://github.com/pivotal-cf/kiln) works by separating out the different requirements of a Tile and "baking" it back into one (useful for very large tiles).
